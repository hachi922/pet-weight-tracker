# ペット体重管理アプリ 🐦

小鳥（または小動物）3匹分の体重・通院記録をつけるシンプルなWebアプリです。  
サーバー不要・ファイルを開くだけで使えます。

## 機能

- **体重記録** — 日付 × 3匹分の体重（g）を表形式で入力（降順表示）
- **グラフ** — 3匹の体重推移を折れ線グラフで表示。各グラフに目標体重ラインを設定可能
- **通院メモ** — 受診日・費用・治療内容を記録
- **テーマカラー** — パステル12色から好みのカラーを選択
- **バックアップ** — JSONエクスポート／インポート（全データ復元）・CSVエクスポート

## 使い方

```
pet-weight-tracker/
├── index.html
├── style.css
└── app.js
```

1. 3ファイルを同じフォルダに置く
2. `index.html` をブラウザで開く

インターネット接続が必要なのは初回のみ（Chart.js と Tabler Icons を CDN から読み込みます）。

## データについて

データはブラウザのメモリ上に保持されます。  
**ページを閉じると消えます**。定期的に「設定 → JSONエクスポート」でバックアップを取ってください。

## GitHub Pages での公開

このリポジトリを GitHub Pages で公開するとスマホからアクセスできます。

1. リポジトリの **Settings → Pages**
2. Source: `Deploy from a branch` → `main` / `/ (root)`
3. 保存後、`https://<username>.github.io/<repo-name>/` で閲覧可能
